# Contribuer

Merci de votre intérêt pour contribuer au projet **Intégration de Solutions Cloud** ! Voici quelques guidelines à suivre pour contribuer :

## Code de conduite
Nous nous engageons à fournir un environnement convivial et accueillant pour tout le monde, peu importe leur origine.

## Comment contribuer
1. Fork le projet
2. Créez votre branche pour la nouvelle fonctionnalité (`git checkout -b ma-fonctionnalite`)
3. Commitez vos changements (`git commit -m 'Ajout d'une nouvelle fonctionnalité'`)
4. Poussez votre branche (`git push origin ma-fonctionnalite`)
5. Ouvrez une Pull Request

Merci encore pour votre intérêt et vos contributions !
